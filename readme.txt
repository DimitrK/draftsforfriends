== Description ==

With Drafts For Friends you can easily create limited time accesible links to your site's draft documents and share 
them with your friends. Get feedback for your posts, pass the content around for corrections without having to grant
your early readers access to your site neither publish a faulty post. 

You are able to delete these sharable links anytime or extend their validity period as needed. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go uner Posts->'Drafts for Friends' to start using the plugin
4. Share some drafts with your friends!
